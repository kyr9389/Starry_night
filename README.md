# 별 헤는 밤 (Starry night)

![banner](https://raw.githubusercontent.com/kyr9389/Starry_night/master/app/src/main/res/drawable/advertise_image.png)

다양한 사용자들의 하루를 공유하는 안드로이드 어플리케이션입니다.

특별하지 않더라도 소중한 당신의 하루를 기록해보세요.

**현재 DB서버 만료로 실행이 불가능합니다.**

## 기술 스택
- JAVA
- PHP
- Gradle
